Java Servlets sind...

  • „Standard“ Java Programme nach Servlet-API (Erweiterung des JDK um JSDK) von SUN

  • Client/Applet <- -> Server/Servlet

  • in der Lage Requests des Clients abzufangen und zu verarbeiten und darauf ein Response zu versenden

  • i.d.L. bei der Verarbeitung alle Fähigkeiten von JAVA auszunützen (RMI, JDBC, UTILS) !!!
